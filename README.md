# ViteDeploy

Современное веб-приложение на базе **React** и **Vite**, использующее **Redux Toolkit** для управления состоянием и **Bootstrap** для стилизации интерфейса. Проект полностью настроен для автоматического деплоя на **GitHub Pages**.

## 🌐 Живая демо-версия

Приложение развернуто и доступно по адресу:  
👉 [https://igogoblin.github.io/viteDeploy](https://igogoblin.github.io/viteDeploy)

---

## 🛠 Технологический стек

*   **Фреймворк и сборка:** [React 18](https://react.dev/) + [Vite 5](https://vitejs.dev/) (с поддержкой ES-модулей)
*   **Управление состоянием:** [@reduxjs/toolkit](https://redux-toolkit.js.org/) & `react-redux`
*   **Стилизация и компоненты:** [Bootstrap 5](https://getbootstrap.com/) & `react-bootstrap`
*   **Уведомления:** `react-toastify`
*   **Линтинг:** [ESLint](https://eslint.org/) (с жесткой конфигурацией на предупреждения)
*   **Деплой:** `gh-pages`
